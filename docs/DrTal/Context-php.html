 DrTal_Context

Default context handling class which stores the variables for the associated template and offers helper functions like a path resolver or escaping functions.

 set

public function set( $name,
$value )

Sets a variable in the context

Arguments

$name The name of the variable to set
$value The desired value for the variable

Note

If the context has been pushed this method will set the variable in the last pushed one.  So the variable will be lost when poping the context.
